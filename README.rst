Tpmrec: A framework of topic model and recommer system
=======================================================

Requirements
------------
The follwoings are required to build Tpmrec:

- Python 2.7+
- NumPy 1.9+


How to build
------------
You can install Tpmrec from the source code. The following command builds Tpmrec.

.. code:: shell

   python setup.py install
